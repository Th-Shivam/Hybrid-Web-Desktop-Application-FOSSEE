# Hybrid Web-Desktop Application (Intern Project)

Simple project structure setup.

## Structure
- `backend/`: Django + DRF
- `web-frontend/`: React app (placeholder)
- `desktop-app/`: PyQt app (placeholder)

## How to Run Backend
1. `cd backend`
2. `pip install -r requirements.txt`
3. `python3 manage.py runserver`

## API
- CSV Upload: `POST /api/upload-csv/`
